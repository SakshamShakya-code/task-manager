# TaskFlow

A clean, dark-themed task manager web app built with HTML, CSS, and Vanilla JavaScript.

## Features
- Add, complete, and delete tasks
- Filter tasks by All / Pending / Completed
- Stats dashboard showing total, completed, and pending count
- Tasks persist after page refresh using localStorage
- Animated splash screen on load
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Live Demo
[View Live](https://sakshamshakya-code.github.io/task-manager/)

## How to Run
Clone the repo and open `index.html` in any browser. No dependencies or installations required.
